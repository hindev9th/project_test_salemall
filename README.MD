# How to setup

```
    1 . composer install
    2 . Import database in folder db/shop.sql
    3 . Config website in file config/Config.php
    4 . Run
```

# Note

### Manager product url
```
    1 . /product
    2 . /product/create
    3 . /product/edit/{id}
    4 . /product/update/{id}
    5 . /product/destroy
```

### Show product url
```
    1 . /
    2 . /product/{id}
```

### Api product url
```
    1 . /api/v1/product/list
    2 . /api/v1/product/get/{id}
    2 . /api/v1/product/update
    2 . /api/v1/product/delete
```
